# deep chatbot

## Introduction

Chatbot created following this [tutorial](https://towardsdatascience.com/how-to-build-your-own-chatbot-using-deep-learning-bb41f970e281)

## Requirements

Python3

## Installation

```bash
python3 -m venv venv
. venv/bin/activate
python3 -m pip install -r requirements.txt
python3 main.py
```

## Execution

```bash
python3 app.py
```
